# About Me
Esmu sysadmin ar vairāku gadu pieredzi serveru, tīklu un infrastruktūras uzturēšanā.
Covid laikā, kad bija mazliet vairāk brīvā laika, aizrāvos ar ethical hacking un penetration testing. Kopš tā laika regulāri mācos un trenējos uz TryHackMe un HackTheBox platformām.
Strādāju galvenokārt ar Kali Linux toolset — nmap, Metasploit, Burp Suite, Gobuster, Feroxbuster, BloodHound, CrackMapExec, sqlmap u.c. Pamazām krāju arī praktisko pieredzi Active Directory vidēs, web app testēšanā.
Pierakstus un walkthrough'us līdz šim turēju Notion, bet šobrīd aktīvi pāreju uz Obsidian + Markdown + Git sync.
Interesē: red teaming, AD exploitation, web & API pentesting, OSINT, un nedaudz arī blue team / detection engineering puse.
